# EBS-state-machine-code
Emergency brake state machine code for the frontbox PCB ensuring all sensors on the car are working properly  as well as digitally filtered and sent in CAN. Also safety statemachine ensuring we can properly build enough hydraulic pressure for emergency braking during autonomous driving. No known bugs and has passed scrutineering in FSG and FSA 2025.
